# Python Programming Course (Bachelor Level)

## 📘 Overview

This repository contains teaching materials for a **complete introductory Python programming course** designed for bachelor-level students.

The course starts from basic computing concepts and gradually progresses to more advanced topics such as:

* Functions
* Classes
* File handling
* Modules
* Code style and best practices

The focus is on **practical learning through examples and exercises**.

---

## 📂 Course Structure

The course is organized into the following modules:

```bash
1 - Introduction to Computing with Python
2 - Expressions
3 - Data Types
4 - Control Statements
5 - Functions
6 - Classes
7 - Exception Handling
8 - File Handling
9 - Modules and Packages
10 - Code Style
```

Each topic includes:

* 📘 **Sample notebooks** → Concept explanations with examples
* 🧪 **Exercise notebooks** → Practice problems for students


The content is organized to support both **teaching** and **self-learning**.

## 🎯 Learning Objectives

By the end of this course, students will be able to:

* Understand core programming concepts
* Write Python programs using variables, data types, and operators
* Use conditional logic and loops
* Create and use functions
* Work with object-oriented programming (classes)
* Handle errors and files
* Organize code using modules
* Follow basic coding standards and best practices

## 🛠️ Setup Instructions

### 1️⃣ Install Python

1. Visit: https://www.python.org/downloads/
2. Download and install the latest version
3. During installation:

   * ✅ Select **"Add Python to PATH"**

Verify installation:

```bash
python --version
```

---

### 2️⃣ Install Visual Studio Code

1. Download from: https://code.visualstudio.com/
2. Install and open VS Code
3. Install extensions:

   * Python (by Microsoft)
   * Jupyter (recommended)

---

### 3️⃣ Running Python Code

#### Option A: Run a Python file

```python
print("Hello, Python!")
```

#### Option B: Use Jupyter Notebook

```bash
pip install notebook
jupyter notebook
```
